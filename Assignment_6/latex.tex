\section{Problem}
An electronic assembly consists of two
subsystems, say, A and B. From previous
testing procedures, the following probabilities are assumed to be known:\\
P(A fails) = 0.2\\
P(B fails alone) = 0.15\\
P(A and B fail) = 0.15\\
Evaluate the following probabilities:
\begin{enumerate}
    \item P(A failsâ€”B has failed)
    \item P(A fails alone)
\end{enumerate}
\section{Solution}
Given, 
\begin{align}
    &P\brak{A \cap B}= 0.15\\
    &P\brak{ B\; fails\; alone} = P\brak B - P\brak{A \cap B} = 0.15\\
    &\implies P\brak B = 0.15+0.15 = 0.30\\
\end{align}
1)  
\begin{align}
    P(A \;fails\;|\;B \;has\; failed) &= \frac{P\brak{A \cap B}}{P\brak B}\\
    &= \frac{0.15}{0.30} = 0.5
\end{align}
2)
\begin{align}
    P\brak{A \; fails \; alone}&= P\brak A - P\brak{A \cap B}\\
    &= 0.2- 0.15 = 0.05
\end{align}
